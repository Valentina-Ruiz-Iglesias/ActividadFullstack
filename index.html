Conectando trayectorias y digitalizando el futuro académico

Sobre el sistema

El Sistema Integral de Gestión Académica Universitaria Digital de la Universidad Raquitich busca centralizar la información académica y administrativa en una sola plataforma web, permitiendo el acceso seguro y en tiempo real para estudiantes, docentes y personal administrativo.

Contexto

La Universidad Raquitich cuenta con múltiples carreras, programas académicos y actores institucionales. Actualmente, la gestión de la información se realiza mediante sistemas dispersos y registros manuales, lo que dificulta la eficiencia y el acceso a los datos.

Problemática

  • Información académica dispersa en distintos sistemas
  • Errores de registro y duplicidad de datos
  • Pérdida de documentación importante
  • Dificultad en el acceso a información en tiempo real
  • Falta de integración entre procesos académicos

Solución propuesta

Se propone el desarrollo de una plataforma web modular basada en una arquitectura de microservicios, capaz de integrar todos los procesos académicos en un solo sistema, mejorando la escalabilidad, mantenibilidad y acceso a la información.

Funcionalidades principales

  • Gestión de estudiantes
  • Gestión académica (asignaturas y calificaciones)
  • Mensajería interna
  • Calendario académico
  • Seguimiento de trayectoria estudiantil
  • Gestión de talleres extracurriculares

Tecnologías utilizadas

  • Frontend: Angular
  • Backend: Spring Boot (Java)
  • Base de datos: MySQL
  • Contenedores: Docker
  • Orquestación: Kubernetes
  • Cloud: AWS

Objetivo del proyecto

Desarrollar una plataforma web que permita centralizar, automatizar y optimizar los procesos académicos, entregando una solución moderna, escalable y eficiente para la gestión universitaria.

Beneficios

  • Acceso a información en tiempo real
  • Reducción de errores administrativos
  • Mejor comunicación entre usuarios
  • Optimización de procesos institucionales
  • Base tecnológica preparada para crecimiento futuro
